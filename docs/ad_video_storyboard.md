# Mapae 광고 영상 스토리보드

## 앱 핵심 기능 요약
- 명함 촬영 → AI OCR 자동 인식 → 디지털 지갑 보관
- 퀵쉐어 (근거리 명함 교환)
- 팀 CRM 파이프라인 (리드→연락→미팅→제안→계약→완료)
- 팀 관리 & 명함 공유
- 3D 명함 뷰어 & 공유

---

## 🎬 버전 A: 30초 광고 (인스타 릴스 / 유튜브 쇼츠)

### 장면 1 (0~3초) — Hook
**화면:** 명함 쌓인 책상 위에서 고통스러운 표정의 직장인 (실사 또는 일러스트)
**텍스트 오버레이:** "명함, 아직도 이렇게 관리하세요?"
**BGM:** 긴장감 있는 짧은 효과음
**전환:** 빠른 줌 인

### 장면 2 (3~7초) — 문제 제기
**화면:** 명함첩 뒤지는 모습 → 찾지 못해 한숨
**텍스트:** "필요할 때 못 찾는 명함"
**나레이션:** "중요한 미팅 전, 명함을 찾을 수 없다면?"

### 장면 3 (7~10초) — 솔루션 등장
**화면:** 스마트폰에 Mapae 앱 로고 등장 (깔끔한 모션)
**텍스트:** "Mapae — 스마트 명함 지갑"
**BGM:** 밝고 경쾌한 음악으로 전환
**나레이션:** "마패가 해결합니다"

### 장면 4 (10~15초) — 핵심 기능 ① 스캔
**화면:** 카메라로 명함 촬영 → AI가 자동으로 이름/회사/연락처 인식 (앱 화면 녹화)
**텍스트:** "찍기만 하면 자동 저장"
**나레이션:** "명함을 찍으면 AI가 자동으로 정보를 인식합니다"
**포인트:** OCR 인식 결과가 빠르게 채워지는 애니메이션 강조

### 장면 5 (15~19초) — 핵심 기능 ② 퀵쉐어
**화면:** 두 사람이 스마트폰을 가까이 대는 장면 → 퀵쉐어 화면 (앱 화면 녹화)
**텍스트:** "만나면 바로 교환"
**나레이션:** "근처 사람에게 바로 명함을 전달하세요"
**포인트:** "주변 검색 중..." → 상대방 발견 → 전송 완료 애니메이션

### 장면 6 (19~24초) — 핵심 기능 ③ 팀 CRM
**화면:** CRM 파이프라인 화면 (리드→연락→미팅→제안→계약→완료)
**텍스트:** "팀 영업 관리까지 한 번에"
**나레이션:** "팀원과 명함을 공유하고, 영업 파이프라인으로 관리하세요"
**포인트:** 전환율 수치가 올라가는 애니메이션

### 장면 7 (24~28초) — 지갑 화면
**화면:** 지갑 화면에서 명함 리스트 스크롤 + 검색 + 카테고리 필터
**텍스트:** "모든 명함, 한 곳에서"
**나레이션:** "검색, 필터, 카테고리로 쉽게 찾으세요"

### 장면 8 (28~30초) — CTA (Call to Action)
**화면:** Mapae 로고 + 앱스토어/플레이스토어 배지
**텍스트:** "지금 무료 다운로드"
**나레이션:** "마패, 지금 시작하세요"
**BGM:** 마무리 효과음

## 🎬 버전 C: 60초 광고 (유튜브 프리롤 / 상세 버전)

### 파트 1: 공감 (0~10초)
- 비즈니스 미팅 후 명함을 받는 장면
- 서랍에 쌓인 명함, 엑셀에 하나하나 입력하는 모습
- "매번 이러고 계신 거 아니죠?"

### 파트 2: 솔루션 소개 (10~20초)
- Mapae 앱 소개
- 명함 촬영 → AI OCR 자동 인식 데모
- "찍기만 하면 이름, 회사, 연락처가 자동으로 저장됩니다"

### 파트 3: 공유 기능 (20~30초)
- 홈 화면에서 3D 명함 뷰어 보여주기
- 퀵쉐어로 근거리 명함 교환
- 링크 공유 기능
- "종이 명함 없이도, 스마트하게 교환하세요"

### 파트 4: 팀 & CRM (30~45초)
- 팀 생성 → 팀원 초대
- 팀 CRM 파이프라인 (리드→연락→미팅→제안→계약→완료)
- 공유 명함에서 CRM으로 가져오기
- "팀 영업 현황을 한눈에 파악하세요"

### 파트 5: 관리 기능 (45~52초)
- 지갑 화면: 검색, 필터, 카테고리 분류
- 즐겨찾기, 상황 태그
- CSV 내보내기
- "필요한 명함을 3초 만에 찾으세요"

### 파트 6: CTA (52~60초)
- 사용자 후기 텍스트 (선택)
- Mapae 로고
- "마패 — 명함의 새로운 기준"
- 앱스토어 / 플레이스토어 다운로드 배지

## 📱 앱 화면 녹화 가이드

광고에 사용할 앱 화면 녹화 시:
1. **해상도:** 최소 1080x1920 (FHD)
2. **프레임:** 60fps 권장
3. **알림바:** 숨기기 (방해금지 모드)
4. **배터리:** 100%로 표시
5. **더미 데이터:** 실제처럼 보이는 샘플 명함 5~10장 준비
6. **동작:** 천천히, 부드럽게 탭/스와이프 (편집 시 1.5배속 가능)
